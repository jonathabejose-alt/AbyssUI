**🔧 Abyss UI — Update Log**

**➕ Added**
- Auto Generator now has two modes: **Blatant** (instant, no cooldown) and **Silent** (5s cooldown to avoid bans)
- Full Bright now has two modes: **Normal** (applies once) and **Loop** (keeps reapplying so teleports don't reset it)
- **Fuse Box TP** — cycles through all fuse boxes on the map
- **Fuse Box ESP** — highlights all fuse boxes in cyan
- **Bear Trap ESP** — highlights all bear traps in orange

**🗑️ Removed**
- **Flight** — removed from Main tab
- **Speed Hack** — removed from Main tab

**⚙️ Changed**
- **Teleport speed** reduced (25 → 10) to avoid bans
- **Noclip** now uses `Heartbeat` instead of `Stepped` for better consistency
